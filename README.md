# MailingServerRohit

To run : npm start




* **URL**

  <_localhost:3001/_>

* **Method:**

   `POST` 
  
*  **Data Params**

   **Required:**
 
   `toList=[array of strings]` array of email id's to send mail to
   
   `subject=[string]` subject of mail
   
   `text=[string]` body of mail

  **example:**
    
    {
    
    "toList":["y2jchandu@mail.com"],
  
    "subject":"from board app",
  
    "text":"hi"
    
    }
